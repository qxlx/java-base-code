我的大学

河北软件职业技术学院1972年建校，直属河北省教育厅，是河北省首家独立建制的以信息技术教育为主的高等职业院校，也是省内最大的信息技术高技能人才培养基地。是河北省唯一一所国家示范性软件职业技术学院建设单位、河北省重点建设示范性高职院校、河北省高职高专人才培养工作水平评估优秀院校、河北省首家单独招生试点单位、首批大学生创业教育示范学校、检查优秀学校。著名计算机专家、中国工程院院士倪光南先生为学院名誉院长。

我的家乡

长子县位于山西省东南部，上党盆地西侧，东距长治市区21公里，北距太原市234公里，距北京670公里。 素有“中国青椒之乡”的美称，当地是中华民族发祥地之一，历史悠久，源远流长，是中国往昔尧王的故里、丹朱的封地、精卫的故乡、西燕的古都。2007年被评为中国千年古县。
